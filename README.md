## ⚙️ Getting Started Locally

### 1. Clone the Repo
```bash
git clone https://github.com/yourusername/LeetGuide.git
cd LeetGuide
````

### 2. Backend Setup

```bash
cd api
npm install
nocemon server.js
```

> 🔐 Create a `.env` file in the `server/` folder:

```env
PORT=5000
GEMINI_API_KEY=your_gemini_api_key
```

### 3. Frontend Setup

```bash
cd client
npm install
npm run dev
```
